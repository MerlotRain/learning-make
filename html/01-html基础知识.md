# 标签

分为单标签和双标签，单标签也叫自结束的标签

```html
<marquee>滚动</marquee> <input />
```

标签之间存在并列关系和嵌套关系

标签的基本属性都卸载标签内部

```html
<input type="text" />
```

```html
<!DOCTYPE html>
<html lang="en">
  <!-- 用于指定语言 -->
  <head>
    <title>hello</title>
    <meta charset="utf-8" />
    <!-- 用于指定字符集 -->
  </head>
  <body>
    <h1>hello</h1>
  </body>
</html>
```
